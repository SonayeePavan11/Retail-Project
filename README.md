# Retail Sales Analysis and Prediction

## Project Overview

This project demonstrates an end-to-end Data Science workflow using a retail sales dataset. The objective is to analyze sales performance, identify business trends, visualize insights, and predict future sales using Machine Learning.

The project includes data preprocessing, exploratory data analysis (EDA), visualization, model building, and performance evaluation.

---

## Objectives

- Load and preprocess retail sales data.
- Perform exploratory data analysis.
- Visualize sales and profit trends.
- Analyze category-wise and region-wise performance.
- Build a Linear Regression model for sales prediction.
- Evaluate the model using standard performance metrics.
- Draw meaningful business conclusions.

---

## Dataset

**File Name:** `Superstore.csv`

The dataset contains the following columns:

- OrderID
- OrderDate
- Region
- Category
- SubCategory
- Sales
- Quantity
- Discount
- Profit
- Segment
